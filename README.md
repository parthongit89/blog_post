# Stillness 🌌
A premium, minimalist sanctuary for deep thought and digital curation.

Stillness is a production-ready, full-stack blogging platform built using Flask and PostgreSQL. It features a modern midnight aesthetic, highly responsive grid layouts, and a robust database infrastructure designed to scale seamlessly.

 🛠️ Tech Stack & Architecture

- **Backend Framework:** Python / Flask
- **Database Engine:** PostgreSQL (Object-Relational Database)
- **ORM & Database Management:** Flask-SQLAlchemy (SQLAlchemy Core)
- **Security & Encryption:** Flask-Bcrypt (Blowfish cipher password hashing)
- **Session Management:** Flask-Login (Secure user authentication states)
- **Frontend Engine:** Tailwind CSS with premium typography (`Capriola`, `SN Pro`, `Jura`)

---

## 🚀 Key Features

* **Premium Minimalist UI:** Designed with an elegant glassmorphism style, modern spacing grids, and custom dark/light color balances.
* **Relational Data Mapping:** Structured schema handling one-to-many relationships between platform Users and their published Posts.
* **Cross-Environment Pipeline:** Customized database configurations allowing seamless data pooling between Linux WSL2 instances and native Windows host engines.
* **Secure Data Layer:** Automatic cryptographic password masking before relational storage commits.
